# 諏訪っ子市場（すわっこいちば）

このサイトは、**諏訪小学校6年生が運営する学習用サイト**です。  
地域の魅力や活動の様子を発信することを目的としています。

---

## サイトについて

「諏訪っ子市場」は、  
諏訪小学校の児童による学習活動の一環として制作・運営されています。

掲載されている内容は、授業や学校行事に基づくものです。
