# 諏訪っ子市場（すわっこ市場）

本リポジトリは、**諏訪小学校6年生が運営する「諏訪っ子市場」**を紹介・宣伝する**公式Webサイト**の**ソースコード**です。

---

## **サイトについて**

* 本サイトは、**「諏訪っ子市場」を地域の方々に知っていただくこと**を目的として制作されています。
* **市場で販売する内容や活動の様子、開催情報**などを発信しています。
* 本サイトは、**学校教育活動の一環として、児童が主体となって運営**しています。
